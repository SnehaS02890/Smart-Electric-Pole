# Smart Electric Pole

## Overview
Smart Electric Pole is an IoT-based system that monitors air quality and vehicle traffic in real time.

## Components
- Arduino Uno
- MQ Gas Sensor
- Ultrasonic Sensor
- LCD Display
- Firebase Realtime Database

## Features
- Vehicle counting
- Air quality monitoring
- Firebase cloud storage
- Real-time alerts

## Technologies
- Arduino IDE
- Python
- Firebase
